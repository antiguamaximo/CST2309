In this project, you will create and deploy a Gym Website with three linked pages:
  • Page1:: About Us (index page)
  • Page 2: Membership Form with Validation
  • Page 3: Display your creativity
You will present the project to the class and explain your design and codes.
Requirements
  • Create a Wireframe using Figma for each page
  • Integrate JavaScript Concepts from Weeks 10 to Week 14
    • Use of JavaScript Functions
    • Use of JavaScript Control Structures: if statements, while and for loops
    • Use of JavaScript Arrays
  • Create and Validate Membership Form with JavaScript 
  • Validate each HTML page using the W3C Markup Validation Service 
  • Integrate Images with JavaScript using a slideshow
  • Integrate CSS
    • Background images
    • Contextual & Attribute Selectors
    • Page Layout: Floats and Grids
  • Use a Responsive Nav Bar
    • Hover Effects
  • Implement Parallax Scrolling
  • Add Google Map with marker
  • Deploy the site on GitHub Pages
  • Complete and upload Final Project Worksheet
  • Create and upload a PowerPoint Presentation that outlines your project
    • Include 3 slides with comparison layout
    • Page should show Figma Wireframe and corresponding web page
  • Add GitHub Pages URL  and the PowerPoint to Blackboard
  • Add GitHub Pages URL  and the PowerPoint to the Final Project Forum in the Discussion Board
